## Evidence checklist before sending
- Original repo URL
- Infringing repo/site URL
- Screenshots of matching pages/files
- Side-by-side code or text comparison
- Dates showing your work existed first
- Specific file URLs or line references

## GitHub-specific note
GitHub's docs say the fastest route is their copyright claims form, and if less than a full repository is infringing, you should identify the specific files or line numbers. GitHub also notes that forks are not automatically disabled, so if forks are infringing too, they should be identified explicitly.
